# Weather Checker Project

## Features
- Console version
- GUI (Tkinter) version
- Web version (Flask)

## Requirements
- Python 3
- `requests` library
- `Flask` for web app

## Run Instructions

### Console
```bash
python weather_checker.py
```

### GUI
```bash
python weather_gui.py
```

### Flask App
```bash
pip install Flask
python app.py
```
Open browser at http://127.0.0.1:5000

## Get API Key
Visit https://openweathermap.org/api to get a free API key.
